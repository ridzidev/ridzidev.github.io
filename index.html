•Cara belajar

Belajar di primitif secara tatap muka, baik online maupun offline.

Online

Pelajar untuk online tidak dibatasi oleh wilayah tertentu, hanya saja dibatasi jumlah pesertanya untuk tiap sesi.

Offline

Pelajar untuk offline dibatasi oleh wilayah tertentu, untuk saat ini hanya tersedia tutor untuk wilayah Banda Aceh dan sekitarnya. Untuk seterusnya wilayah akan diperluas tergantung ketersediaan tutor di wilayah tersebut. jumlah peserta dibatasi untuk tiap sesi.

•Metode dan langkah pembayaran
Untuk paket 1x Pertemuan offline

langsung membayar kepada tutor saat pembelajaran berlangsung.

Untuk paket lainnya

dapat mengikuti langkah berikut:
1. Pesan guru private (halaman utama)
2. Konfirmasi oleh Primitif learning
3. Kirim foto bukti transfer
4. Mendapatkan username dan password (Untuk kali pertama daftar)
5. Login untuk melihat sisa pertemuan
6. Klaim tanggal pertemuan berikutnya
