Penjelasan Sekilas

Math command adalah alat untuk melakukan perhitungan matematika sehari-hari. dirancang untuk siswa, mahasiswa, guru, dosen, dll. Hasil perhitungan dapat dilihat tepat dibawah ekspresi setelah di enter. Jika terjadi perubahan diatas, maka semua perhitungan terkait dibawahnya juga akan berubah

Bilangan

Math Command mendukung bilangan pecahan desimal dan dapan dinyatakan dalam notasi ilmiah.

Bilangan Kompleks

Bilangan kompleks dapat dibuat dengan satuan imajiner i.

Fungsi dan Operator

Math Command mendukung semua fungsi biasa (seperti sin, sqrt, exp) dan operator (seperti a + b, a - b, a * b, a / b). bagian dari ekspresi dapat dinyatakan dalam tanda kurung (...).

Sangat mungkin untuk membuat fungsi sendiri dengan 0,1,2, atau banyak variable.

Grafik fungsi

Grafik fungsi dapat digambarkan dengan fungsi plot. Grafik dapat digerakan atau diperbesar dengan menggeser dan menggulirkan grafik. Grafik dapat digambarkan dengan satu atau banyak fungsi sekaligus. Anda dapat menggunakan variabel secara bebas seperti x,t, dll.

Matriks

Math Command mendukung matrik dimensi-n. Matrik dibentuk dengan tanda kurung siku. Nilai pada baris dipisahkan dengan tanda koma (,) dan nilai pada kolom dipisahkan dengan semicolon (;) Suatu subset dari sebuah matriks dapat diambil dengan indeks diantara kurung siku matrix[...], dengan cara yang sama, suatu subset dapat diganti dengan , matrix[...] = [...]

Perlu diperhatikan bahwa semakin besar suatu matriks maka semakin lama proses perhitungannya.

Satuan

Math Command mendukung satuan. Satuan dapat digunakan dalam aritmatika dasar, operasi seperti tambah dan kurang. Semua satuan dapat dilihat disini.

Kata

Kata dapat dibentuk dalam double quotes ".

Komentar

Komentar dapat ditambahkan untuk menjelaskan perhitungan dalam bentuk teks. Komentar dapat ditulis dengan menggunakan tanda pagar #. Komentar juga dapat ditambahkan disamping ekspresi perhitungan.

Variabel dan konstanta

Math Command mendukung konstanta matematika sepertipi dan e.

Variabel baru dapat didefinisikan dengan tanda sama dengan =.
